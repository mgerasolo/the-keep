## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gaps, inline padding in tree nodes |
| sm | 8px | Compact element spacing, tree node vertical padding |
| md | 16px | Default element spacing, sidebar section padding |
| lg | 24px | Panel padding, content margins |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Welcome screen vertical spacing |
| 3xl | 64px | Not used in Phase 1 |

Exceptions: none for Phase 1. All UI is desktop-only (no touch targets required -- private network app).

Source: Standard 8-point grid (default)

## Typography

| Role | Size | Weight | Line Height | Font | Usage |
|------|------|--------|-------------|------|-------|
| Small | 12px | 400 (regular) | 1.33 | Geist Sans | Breadcrumbs, metadata labels, status bar text |
| Body | 13px | 400 (regular) | 1.54 | Geist Sans | Tree node labels, tab labels, search results, UI text |
| Content | 14px | 400 (regular) | 1.71 | Geist Sans | Markdown body text in viewer, dialog body text |

**Code typography:**

| Role | Size | Weight | Line Height | Font | Usage |
|------|------|--------|-------------|------|-------|
| Code inline | 13px | 400 | 1.54 | Geist Mono | Inline code in markdown |
| Code block | 13px | 400 | 1.6 | Geist Mono | Fenced code blocks with syntax highlighting |

## Color

All colors use CSS custom properties defined by shadcn's dark theme. Values below are the semantic tokens.

| Role | CSS Variable | HSL Value | Usage |
|------|-------------|-----------|-------|
| Dominant (60%) | `--background` | 240 10% 3.9% | Page background, main content area |
| Secondary (30%) | `--muted` | 240 3.7% 15.9% | Sidebar background, card surfaces, code block backgrounds |
| Accent (10%) | `--primary` | 217.2 91.2% 59.8% | See reserved-for list below |
| Destructive | `--destructive` | 0 62.8% 30.6% | Delete actions only |

**Additional semantic colors:**

| Token | HSL Value | Usage |
|-------|-----------|-------|
| `--foreground` | 0 0% 98% | Primary text on dark backgrounds |
| `--muted-foreground` | 240 5% 64.9% | Secondary text: tree node icons, placeholders, metadata |
| `--border` | 240 3.7% 15.9% | Panel dividers, input borders, card borders |
| `--ring` | 217.2 91.2% 59.8% | Focus rings on interactive elements |
| `--accent` | 240 3.7% 15.9% | Hover state on tree nodes, tab hover |
| `--accent-foreground` | 0 0% 98% | Text on hover surfaces |

**Accent (--primary, blue) reserved for:**
1. Active tab bottom border indicator
2. Selected tree node background highlight
3. Search match text highlights in results
4. Focus rings on inputs and interactive elements
5. Active/pressed button states
6. Search palette active result highlight

**Accent is NOT used for:** general text, borders at rest, icons at rest, section headers, or decorative elements.

**Folder icon color:** `hsl(217 70% 60%)` -- blue tint matching accent, applied to Folder/FolderOpen lucide icons only.

Source: shadcn dark theme defaults (RESEARCH.md confirms shadcn/ui + Tailwind 4). Cursor uses a similar blue-accent-on-dark palette.

## Component Interaction Contracts

### File Tree (SHELL-01, SHELL-02, SHELL-09)

| Interaction | Behavior |
|-------------|----------|
| Single click file | Opens file in tab (or activates existing tab) |
| Single click folder | Toggles expand/collapse |
| Right-click any node | Opens context menu |
| Drag file to folder | Moves file (Server Action) |
| Drag feedback | Ghost node follows cursor, valid drop target highlighted with accent border |
| Keyboard: Enter | Opens file / toggles folder |
| Keyboard: F2 | Inline rename |
| Keyboard: Delete | Opens delete confirmation dialog |
| Keyboard: Arrow keys | Navigate tree nodes |
| Inline rename | Input replaces node name, Enter to confirm, Escape to cancel |
| Loading state | Skeleton shimmer on tree area during initial load |

### Tab Bar (SHELL-03, SHELL-04, SHELL-05)

| Interaction | Behavior |
|-------------|----------|
| Click tab | Activates tab, loads file content |
| Click close (X) | Closes tab, activates nearest neighbor |
| Middle-click tab | Closes tab |
| Right-click tab | Context menu: Close, Close Others, Close All |
| Close last tab | Shows empty/welcome state |
| Tab overflow | Horizontal scroll with arrow indicators (no wrapping) |
| Active tab indicator | 2px bottom border in accent color |
| Tab max width | 180px with text truncation via ellipsis |
| Tab icon | File icon (lucide `File`) at 14px, muted-foreground color |

### Search Palette (SEARCH-01, SEARCH-02, SEARCH-03)

| Interaction | Behavior |
|-------------|----------|
| Trigger | Cmd+K or click search icon in sidebar |
| Input | Auto-focused, searches on type with 200ms debounce |
| Results | File name bold, matching context below in muted-foreground |
| Click result | Opens file in tab, closes palette |
| Keyboard: Arrow keys | Navigate results |
| Keyboard: Enter | Opens highlighted result |
| Keyboard: Escape | Closes palette |
| No results | "No results found" with suggestion to try different term |
| Result limit | Show top 20 matches, sorted by relevance |

### Delete Confirmation Dialog (SHELL-08)

| Interaction | Behavior |
|-------------|----------|
| Trigger | Delete from context menu or Delete key |
| Dialog | Modal with title, body (file/folder name in bold), Cancel + Delete buttons |
| Delete button | Styled with destructive variant (red background) |
| Cancel | Closes dialog, no action |
| Escape | Closes dialog, no action |
| Click outside | Closes dialog, no action |
| After delete | Close associated tab if open, refresh tree |

### File Create Dialog (SHELL-06)

| Interaction | Behavior |
|-------------|----------|
| Trigger | "New File" button or context menu > New File |
| Dialog | Modal with input for file name, Cancel + Create buttons |
| Input | Auto-focused, placeholder "untitled.md" |
| Validation | Prevent empty names, show error for duplicate names |
| After create | Open new file in tab, expand parent folder in tree |

### Markdown Viewer (VIEW-01, VIEW-02, VIEW-03)

| Interaction | Behavior |
|-------------|----------|
| Frontmatter display | Distinct block above content: mono font, key:value layout, muted background, bordered |
| Code blocks | Syntax-highlighted via shiki, github-dark theme, line numbers visible |
| Line numbers (code) | Right-aligned in gutter, muted-foreground color, not selectable |
| Content text | 14px Geist Sans, standard prose styling (headings, lists, links) |
| Links | Accent color, underlined on hover |
| Tables | Bordered, alternating row backgrounds using muted color |
| Scroll | Vertical scroll for long content, sticky tab bar above |
