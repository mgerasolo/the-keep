# Getting Started with The Keep

Welcome to your personal knowledge base.

## Quick Start

1. **Navigate** using the sidebar file tree
2. **Create** new notes with `Cmd+N`
3. **Search** with `Cmd+K`
4. **Link** notes using `[[note-name]]` syntax

## Markdown Features

### Headers

Use `#` through `######` for heading levels.

### Code Blocks

```typescript
const greeting = (name: string) => {
  return `Hello, ${name}!`
}
```
