# The Keep

Personal knowledge management system inspired by Obsidian, built with Next.js.

## Overview

A local-first markdown editor with:
- File tree navigation
- Split pane editing
- Full markdown support with syntax highlighting
- Command palette (Cmd+K)

## Stack

- **Frontend:** Next.js 16, React 19
- **Styling:** Tailwind CSS, shadcn/ui components
- **State:** Zustand
- **Editor:** Custom markdown renderer with rehype-pretty-code
